# Sonorie

Sonorie é um reprodutor de música offline moderno criado com Kotlin, Jetpack Compose, Material 3 e Media3/ExoPlayer.

## Versão atual

Sonorie v0.2.1 — Notification Sync

## Recursos já validados

- Interface Android real com Material 3 / Material You
- Permissão de áudio local
- Leitura de músicas locais via MediaStore
- Biblioteca offline
- Reprodução com ExoPlayer
- Mini-player
- Busca na biblioteca
- Tela Player
- Botões anterior/próxima
- Progresso básico da faixa
- Serviço dedicado de reprodução
- Notificação de mídia
- Controles na tela de bloqueio
- Sincronização inicial entre serviço/notificação e UI do app
- Build APK via GitHub Actions
